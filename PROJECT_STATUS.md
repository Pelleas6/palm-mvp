# État actuel du projet

Ce dépôt héberge désormais **Le Pouls du Monde**, un observatoire visuel de signaux médiatiques mondiaux.

La documentation de référence est le [README](./README.md).

Les fichiers `HANDOFF_PALM_MVP*.md` et `HANDOFF_STANDBY_OTP.md` décrivent un ancien prototype de lecture de main. Ils sont conservés comme historique technique et ne décrivent plus l'application active.

## Application active

- accueil : carte mondiale et flux d'articles ;
- données : RSS publics, GDELT Web N-Grams, cache côté serveur ;
- interface : thème sombre "atlas vivant", navigation et footer cohérents ;
- compteur : total discret d'explorations, rendu non bloquant ;
- contrôle : tests unitaires et build Next.js.
